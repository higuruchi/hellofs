# hellofs

A local file system where the string HelloWorld is read every time the read system call is executed.

## Requirement

- Ubuntu == 10.04
- Linux Kernel == 2.6.32.x

## Build

```
$ make
```

## Install

```
$ sudo make install
$ sudo make load
```
